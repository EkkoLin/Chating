# Project 4 - *Chating*

**Chating** is a basic twitter app to read and compose tweets the [Twitter API](https://apps.twitter.com/).

Time spent: **3** hours spent in total

## User Stories

The following **required** functionality is completed:

- [x] User can sign in and sign up account
- [x] User can view and send messages
- [x] User can logout from the message page
